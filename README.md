# AI Systems Builder

## Overview
This project is an AI-powered system builder that allows users to interact with an intelligent assistant, automate tasks, and generate outputs using LLMs.

## Features
- Chat-based interface
- AI-powered responses
- Modular system design
- Scalable architecture (WIP)

## Tech Stack
- React (Frontend)
- Vite
- (Planned) FastAPI / Node backend
- (Planned) LLM integration

## Architecture
User → Frontend (React) → API → AI Model → Response

## How to Run
npm install
npm run dev
